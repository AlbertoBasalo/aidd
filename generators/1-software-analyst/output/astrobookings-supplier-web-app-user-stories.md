# AstroBookings: Supplier Web Application User Stories

> Generated on: Monday, August 12, 2024, 21:45 PM UTC

## Roles

- `Supplier`: Representative of a company offering space launches

## User Stories

### Authentication

1. As a `Supplier`, I want to **log in to my account** _so that I can access the supplier portal_.

### Rocket Management

2. As a `Supplier`, I want to **add new rockets** _so that I can offer them for launches_.

3. As a `Supplier`, I want to **update rocket details** _so that I can keep the information accurate_.

4. As a `Supplier`, I want to **view a list of my rockets** _so that I can manage my fleet_.

### Launch Management

5. As a `Supplier`, I want to **schedule new launches** _so that I can offer space trips to customers_.

6. As a `Supplier`, I want to **update launch details** _so that I can provide accurate information to customers_.

7. As a `Supplier`, I want to **abort a scheduled launch** _so that I can manage unforeseen circumstances_.

8. As a `Supplier`, I want to **view a list of my scheduled launches** _so that I can track my offerings_.

### Booking Management

9. As a `Supplier`, I want to **view bookings for my launches** _so that I can issue an invoice_.

10. As a `Supplier`, I want to **see detailed passenger information** _so that I can ensure compliance with regulations_.
