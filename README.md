# AI Driven Development

Prompt, rules and procedures for AI Driven Development

[Repository](https://github.com/AlbertoBasalo/aidd)

## Features

- [rules](./rules/) system prompt to be used as guidelines
  - (for cursor copy and paste in your project as `.cursorrules`)
  - or use the as prompts for your own agents
- [procedures](./procedures/) instructions for code agents
  - to be used as is or
  - as a context for other prompts

## Blogs

If interested in more details about how this works, you can read the following blogs:

- [AiddBot.com in English](https://aiddbot.com)
